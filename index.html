Presentation.Java: WebアプリケーションのプレゼンテーションをサポートするJavaプロダクト

プロダクト 説明 Resources
Cubby Java/Servlet2.4/JSP.2.0 をベースにしたシンプルな Web アプリケーションフレームワーク
Mayaa HTMLをベースとしたテンプレートによるWEBフロントサービスエンジン
S2BlazeDS BlazeDS(Flex/AIRのサーバサイドサービス)とSeasar2を連携させるフレームワーク
S2Flex FlashプレーヤーからFlexを用いてS2Containerのコンポーネントを呼び出すライブラリ
S2JSF HTMLをベースとしたテンプレートエンジンを提供するJSFベースのフレームワーク
S2OpenAMF FlashプレーヤーからFlash Remotingを用いてS2Containerのコンポーネントを呼び出すライブラリ
S2Portlet S2Container を JSR 168 準拠のポートレット上で利用するためのライブラリ
S2Struts StrutsからS2Containerのコンポーネントを呼び出すライブラリ
SAStruts Super AgileにStrutsとSeasar2で開発するためのフレームワーク
Teeda JSFの実装とDIとAOPの機能をベースとした拡張機能を提供するフレームワーク
Ymir フィルタ指向のシンプルなWebアプリケーション構築フレームワーク

Persistence.Java: 永続化をサポートするJavaプロダクト

プロダクト 説明 リソース
Kuina JPA (Java Persistence API) フレームワーク
S2Dao XMLによるマッピング定義や実装コードを必要としないO/R Mapping フレームワーク
S2Hibernate HibernateをS2Containerのトランザクション制御と組み合わせて使うためのライブラリ
S2JDBC 流れるようなインターフェースと 2Way-SQL をサポートした O/R Mapping フレームワーク
(Seasar2の一部です)
S2OpenJPA OpenJPAをS2Containerのトランザクション制御と組み合わせて使うためのライブラリ
S2TopLink TopLinkをS2Containerのトランザクション制御と組み合わせて使うためのライブラリ

Communication.Java: 分散処理をサポートするJavaプロダクト

プロダクト 説明 リソース
S2Axis Axisを用いてS2Containerのコンポーネントをリモート呼び出しするためのライブラリ
S2JCA JCA (J2EE Connector Architecture) 準拠のリソースアダプタを使用するための JCA Application Server Facilities 実装
S2JMS 非同期メッセージングを容易に送受信するためのライブラリと、受信メッセージをS2Containerのコンポーネントにバインディングするフレームワーク
S2Remoting S2Containerでリモート呼び出しを行うための共通基盤ライブラリ
S2RMI RMIを用いてS2Containerのコンポーネントをリモート呼び出しするためのライブラリ

Tools.Java (Eclipse更新サイト): Javaプロダクト用の開発ツール

プロダクト 説明 リソース

S2JSFプラグイン S2JSFのページテンプレート(HTML)を編集・検証するためのEclipseプラグイン
