S2 software which are Eclipse plugin may be installed/updated from the following sites.

Eclipse 3.0.x update site (Available plugin: Kijimuna，S2JSF)
http://www.seasar.org/eclipse/updates/3.0/
Eclipse 3.1.x update site (Available plugin: Kijimuna)
http://www.seasar.org/eclipse/updates/3.1/

Be sure to select the correct site for your version of Eclipse.
Kijimuna supports both Eclipse version 3.0 and version 3.1.
Currently, S2JSF plugin only support Eclipse version3.0.

S2 plugin may be install/updated by the following steps:
For a screen shots of these steps, refer to the tutorial page.

  1. Open Eclipse
  2. From Eclipse menu, select "Help" -> "Software Updates" -> "Find and Install..."
  3. "Install/Update" dialog is displayed. Select "Search for new features to install" option and select the "Next" button
  4. In the Install dialog, select the "New Remote Site..." button and select the "Next" button
  5. "New Update Site" dialog is displayed. In the "Name" field, enter "The Seasar Project" and in the "URL" field enter the URL listed above depending on the version of Eclipse you are using. Select the "OK" button.
  6. "The Seasar Project" should be listed in the "Sites to include in search:" list. Check the Seasar Project checkbox and select the "Next" button.
  7. In the "Select the features to install" list, check either or both "Kijimuna" and "S2JSF plugin" and select the "Next" button.
  8. Read the license and check "Agree with the following Terms and Conditions" if you agree with the license. Select the "Next" button
  9. Select the "Finish" button to start the download and installation
