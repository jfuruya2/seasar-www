Description

s2dotnet is a port of Java version Seasar2 to .NET framework.

Requirements

s2dotnet requires the followings:

.NET Framework
2.1.11 or later
Transactions are properly processed on Windows2000, Windows XP Home, Windows XP Professional, and Windows Server 2003.
