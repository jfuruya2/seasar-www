Introduction

S2Dao.NET is a object relational mapping framework that does not use XML to map relations. It is a port of Java S2Daoを.NET to Framework 1.1. S2Dao.NET is written in C# but it may be used from VB.NET as well.

S2Dao.NET has the following features that assists to improve data access development productivity.

  • Require minimal knowledge of ADO.NET
  • Automatic generation of SQL commands
  • Able to execute SQL commands written in a SQL file (2 Way SQL)
  • Easy to learn
Documents are still in development. Until they are ready, please refer to Java S2Dao documents. S2Dao.NET supports custom attributes instead of Java annotations that are available in Java version of S2Dao.
