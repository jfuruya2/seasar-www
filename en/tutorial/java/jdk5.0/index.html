JDK 5.0 Installation by H.Ozawa

  1. Download Sun JDK5.0 from Sun Java J2SE 5.0 Download page.
  2. Execute the downloaded file jdk-1_5_0_05-windows-i586-p.exe.
  3. Select I accept the terms in the license agreement radio button and select the "Next >" button.
  4. The default installation directory is in the "Program Files" directory in system drive (In the sample screen below, Windows is installed in the "D" drive. In most system, this will be the "C" drive.
    I prefer to install Java directly under the root directory.
    To change the default installation directory, select the Change... button.
  5. Change "Folder name" to directory create "Java" directory in the root directory by deleting "Programs Files". Select the OK button.

  6. Verify that the Install to: directory has changed.
    Select the "Next >" button.

  7. Installation will begin. This will take few minutes depending on your computer.

  8. After J2SE Development Kit 5.0 is installed, J2SE Runtime Environment 5.0 is installed.
    Change the Install to: directory to match the J2SE Development Kit 5.0 directory.
    Select the Change button.

  9. Delete "Programs Files" from the Folder name: and select the OK button.

  10. Verify that the Install to: folder has changed and select the Next button.

  11. Uncheck Microsoft Internet Explorer to not install Java plug-in in Microsoft Internet Explorer.
    Select the Next > button.

  12. J2SE Runtime Environment will be installed. This may take several minutes depending on your computer.

  13. After the installation is completed, select the Finish button.

  14. After installing JDK and JRE, it is convenient to set the environmental variable so we will be able to execute Java directly from the command console.
    Select the Start menu and select Settings-Control Panel.

  15. Select the System icon.

  16. Select the Advanced tab and select the Environment Variables... button.

  17. Select Path in the System variables and select the Edit... button.

  18. Append the folder where you've installed J2SE Development Kit 5.0 + \bin.
    Select the OK button.

  19. Select the OK button.

  20. Select the OK button.

  21. Let test if Java is properly installed.
    Select the Start menu button and select Run....

  22. Type in cmd to open a command console window.
    Select the OK button.

  23. In the command console window, type in java -version and press the Enter key to show the version of Java that is installed on your computer.
    It should be build 1.5.0.
    This completes Java installation.


    Continue with setup for the tutorial.
