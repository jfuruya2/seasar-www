Seasar2 Installation

It's finally time to install Seasar2. To use Seasar, source codes are not absolutely necessary but since Seasar2 does not yet come with JavaDoc, it is very benefitial to import the source code to find parameters and datatypes.
  1. First of all, download Seasar2 from the Seasar site.
    Select File - Import... in the Eclipse menu.

  2. Select Existing Projects into Workspace.
    Select the Next > button.

  3. Select the Select archive file: radio button and select the Browse... button.

  4. Move to the folder where Seasar archive file was downloaded and select the Seasar archive file.(In the example screen below, S2.2.10.zip is the most recent Seasar archive file.)
    Select the Open button.

  5. Check both the seasar2 and seasar-www and select the Finish button.
    Files will be imported - this may require a little bit of time.

  6. Make sure seasar2 and seasar-www are listed in the Package Explorer.

  7. There is a red "x" next the seasar2 indicating some error. This is due to Kijimuna plugin detecting seasar syntax error. Since we know that it is not an error, we'll turn Kijimuna off.
    Select seasar2 with your mouse.

  8. Select Project - Properties from the menu.

  9. Select Kijimuna on the left hand side and uncheck the Seasar2 project.
    Select the OK button.
