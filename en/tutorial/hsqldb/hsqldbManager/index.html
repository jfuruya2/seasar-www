Using HsqldbManage

These steps will create a simple table by executing a sample script included with HSQLDB.
  1. From File Explorer, select the data folder under hsqldb.
    Double click on file runManager.bat.

  2. A Connect dialog box will appear.
    Select HSQL Database Engine Server as the Type, and enter jdbc:hsqldb:hsql://localhost:9001 as the URL.
    Select the Ok button.

  3. A HSQL Database Manager window will appear.
    To execute a script, select File - Open Script... from the menu.

  4. Select the file sampledata.sql in the sample folder.

  5. The script is too long to be viewable in the limited script entry space and displays This huge file cannot be edited. Please execute.
    Select the Execute button.

  6. In the result area, 0 is displayed under the update count column.

  7. To view the created table, select View - Refresh Tree from the menu.

  8. CUSTOMER, INVOICE, ITEM, PRODUCT should appear on the navigator area.

  9. Let's try shutting down the HSQLDB server using the HSQLDB Manager.
    Select Command - SHUTDOWN from the menu2.

  10. SHUTDOWN command is entered into the command area and syntax of the shutdown command in shown in the result area.
    Select the Execute button.

  11. To exit HSQLDB Manager, select File - Exit from the menu.
