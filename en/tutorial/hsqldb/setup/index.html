Hsqldb Installation

Hsqldb installation is very easy. Just extract the file in the archive. That's it!
Following shows steps to startup the database server.
  1. Let's create new folder to hold database files.
    Select File - New - Folder.
  2. Let's name the folder data. It seems Hsqldb script files use data.

  3. Create server.properties file in folder data with the following script.
    The script tells Hsqldb to name the database file demo and use port 9001 for management access (this this the default port number).
    server.properties 
    server.database=demo
server.port=9001


  4. It is possible to start and maintain Hsqldb by typing Java commands but since that's tiresome, let's just copy them from the demo folder.
    Files to copy are:
    1. runManager.bat - GUI utility to execute SQL commands
    2. runManagerSwing.bat - Same as runManager.bat but with Swing
    3. runServer.bat - Script to start Hsqldb database server
    4. runUtil.bat - Script to use Hsqldb utilities
    5. runWebServer.bat - Script to start Hsqldb web server
    Select these file with your mouse. Press the Ctrl key while clicking the left mouse button to select multiple files at once.

  5. Keep pressing the Ctrl and the left mouse button while dragging the mouse pointer to the data folder. Release the mouse to copy the files.

  6. Move to the data folder and check that the files were copied.

  7. It's finally time to start up the Hsqldb server!
    Select the runServer.bat file.

  8. A console window should open and display database server start up process.
    Do not close this window! Proceed to the next page to properly shutdown the server.
    If the console window closes, it means there is something wrong. Recheck of each step.
