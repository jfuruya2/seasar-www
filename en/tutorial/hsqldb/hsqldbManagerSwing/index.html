Using HsqldbManagerSwing

These steps will execute a test sql script.
  1. From File Explorer, select the data folder under hsqldb.
    Double click on file runManagerSwing.bat.

  2. A Connect dialog box will appear.
    Select HSQL Database Engine Server as the Type.
    Select the Ok button.

  3. Let's execute a test script.
    From the menu, select Command - Test Script.

  4. A test script will be shown in the Command area.
    Select Execute SQL from the icon menu.

  5. Result of executing the script is shown in the result area.

  6. To shutdown the HSQLDB server, select Command - SHUTDOWN from the menu.

  7. SHUTDOWN command is displayed in the command area and the syntax of the SHUTDOWN command is displayed in the result area.
    Select the Execute SQL from the icon menu area.

  8. To exit HSQL Database Manager, select File - Exit from the menu.
