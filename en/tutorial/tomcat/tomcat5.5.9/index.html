Tomcat 5.5.9 Installation

Tomcat is not actually required to use Seasar but I've included it here because we will be using it to build Web applications using Seasar.
  1. Download Tomcat from the Apache Tomcat site.
  2. Execute the downloaded file jakarta-tomcat-5.0.26.zip.
  3. Select the "Next >" button.

  4. Read the License Agreement and select the I Agree button.

  5. To include Tomcat example files, check the Example and Webapps select the Next > button.

  6. Check the Destination Folder and select the Next > button.

  7. Enter the Password and select the Next > button.

  8. Check if the path where J2SE 5.0 JRE is correct and select the Install button.

  9. Tomcat files will be installed.

  10. Check Run Apache Tomcat to verify Tomcat was properly installed and select the Finish button.

  11. Wait until Tomcat starts.

  12. Open a web browser and enter http://localhost:8080 as the Address.
    If Tomcat was properly installed, the following page will appear.
