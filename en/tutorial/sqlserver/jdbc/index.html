MS SQL Server JDBC Driver Installation

These steps are only necessary when using MS SQL Server.
  1. Goto the the MS Downloads page to download JDBC driver for the MS SQL Server.
  2. Enter the following information in the respective text entry fields:
    Keywords(optional): jdbc
    Product/Technology: SQL Server
  3. Execute the search by select the Go button.

  4. A list of links containing keywords jdbc and SQL Server will show.
    Select SQL Server 2000 Driver for JDBC.

  5. Select setup.exe from the download section. This is the JDBC driver.

  6. Execute the downloaded setup.exe file.
