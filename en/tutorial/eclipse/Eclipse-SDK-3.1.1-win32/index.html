Eclipse-SDK-3.1.1-win32 Installation by H.Ozawa

Eclipse is used to develop Seasar and most Seasar packages uses Eclipse as a development platform. Eclipse if not required but highly recommended.
  1. Download Eclipse from the Eclipse site.
  2. Extract the downloaded file eclipse-SDK-3.1.1-win32.zip to a folder.This folder will the Eclipse installation folder. There is not extracting files are running setup with Eclipse.
    I prefer to extract it directly under the root folder.
  3. This isn't absolutely necessary, but I like to have a shortcut on my desktop.
    Create a shortcut on the desktop by selecting the eclipse.exe file with your mouse and pressing the ALT key and drag the file to your desktop.

  4. Let's try starting it up! Double click on the Shortcut we just created. A "Workspace Launcher" dialog like the one below should appear. Enter a folder where all development files are to be created. In this example, we'll create a folder C:\Projects\seasar\ to hold all our example files.
    Select the Browse... button.

  5. Select the New Folder and enter Projects as the new folder name.


  6. Select the New Folder again and enter seasar as the new folder name this time.


  7. Select the OK button.


  8. After making sure the new folder name appears in the "Workspace", select the OK button.


    9. After selecting a folder, select the OK button.

  9. If all went well, Eclipse screen will appear. Click on the X to close the panel.


  10. We are now ready to start writing programs!
