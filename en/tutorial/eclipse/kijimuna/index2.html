Kijimuna Plugin Installation by H.Ozawa

Manually installing Kijimuna is a little more complicated. These are the steps.
  1. Download Kijimura Plugin from the Seasar site.
  2. Extract files from the downloaded file kijimuna0.6.26_for_eclipse3.1.zip.
  3. Extract org.seasar.kijimuna.core_0.6.26.jar and org.seasar.kijimuna.ui.6.26.jar in the plugins folder within the downloaded file.
    It is easier to use graphical archive tool like Winzip to unzip jar files, but it can also be done from the DOS prompt with jar -x command.

  4. Change to plugins folder under where Eclipse was installed.
  5. Create 2 new folders:
    org.seasar.kijimuna.core_0.6.26.jar
    org.seasar.kijimuna.ui.6.26.jar

  6. Move extracted content of jar files extracted in step 3 to under the folder of there names.
