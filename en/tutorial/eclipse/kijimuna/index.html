Kijimuna Plugin Installation by H.Ozawa

Kijimuna is an Eclipse plugin to validate against Seasar syntax. Without Kijimuna, we would have to look at Java compilation errors to find syntactical errors.
The easiest way to install Kijimuna is from Eclipse. But if you don't have an Internet connection, you're still able to install Kijimuna from the downloadable archive file by following this steps.
  1. Start Eclipse and select Help - Software Updates - Find and Install...

  2. Select Search for new features to install and select the Next > button.

  3. Select the New Remote Site... button.

  4. Enter Kijimuna as the Name and enter http://www.seasar.org/eclipse/updates/3.1beta/ as the URL. Select the OK button.

  5. Check the Kijimuna as the site to include in search and select the Finish button.

  6. Check Kijimuna as feature to install and select the Next > button.

  7. Select the I accept the terms in the license agreement and select the Next > button.

  8. Make sure the directory Eclipse is installed in displayed as the Install Location. If it is different, select the "Change Location..." button to select the folder where Eclipse was installed. Select the Finish button.

  9. Necessary files will be downloaded. Do not select any buttons in this dialog box.

  10. Select the Install All button.

  11. Select the Yes to restart Eclipse.

  12. After Eclipse is restarted, let's check if Kijimuna was properly installed.
    Select Windows - Preferences...

  13. Verify that Kijimuna is listed.
    Select the Cancel button.
    This completes Kijimuna plugin installation.
